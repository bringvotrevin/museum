# 국립중앙박물관(NMK) 적정 입장료 산정

글로벌 벤치마킹, 관람객 감성분석, 수요 시뮬레이션을 통해 국립중앙박물관의 데이터 기반 적정 입장료를 도출하고, 수익과 공공성을 모두 고려한 요금 제안

## 프로젝트 배경

국립중앙박물관은 세계 방문객 순위 8위(2024년 기준), 전시면적 27,090㎡(세계 6위), 소장품 약 42만 점을 보유한 세계적 박물관이지만 입장료는 무료로 운영되고 있습니다. 재정 자립도가 낮아 운영비의 대부분을 정부 예산에 의존하는 상황에서, 글로벌 주요 박물관들이 유료 운영으로 전시 품질과 재정 지속성을 확보하는 추세와 대조적입니다. 이 프로젝트는 데이터를 기반으로 합리적인 입장료 수준을 도출하는 것을 목표로 합니다.

## 분석 흐름

```
Step 1. 글로벌 벤치마킹 — K-means 클러스터링 + 다중 선형 회귀
        → NMK 규모 기준 적정 입장료 $12.20(₩17,080) 도출

Step 2. 관람객 감성 평가 — 사전 기반 감성분석 + LDA 토픽 모델링
        → 무료 박물관 감성(0.804) > 유료(0.661), 유료화 리스크 정량화

Step 3. 수요-수익 시뮬레이션 — Prophet 시계열 예측 + 선형 수요 모델
        → 시나리오 비교 및 최종 요금 정책 제안
```

## 주요 결과

### 교차검증
- Step 1 회귀분석 적정가: **$12.20 (₩17,080)**
- Step 3 수익 시뮬레이션 최적가: **₩17,595**
- 두 독립적 분석이 동일 가격대로 수렴 → 신뢰도 확보

### 시나리오 비교

| 시나리오 | 내국인 | 외국인 | 연간수익 | 방문객 변화 |
|---|---|---|---|---|
| A. 단일 요금제 | ₩17,595 | ₩17,595 | 약 519억 | ▼ 34.8% |
| B-1 수익 최적 이중가격 | ₩17,595 | ₩17,595 | 약 519억 | ▼ 34.8% |
| B-2 공공성 우선 이중가격 | ₩5,000 | ₩17,595 | 약 271억 | ▲ 9.8% |
| **최종 제안** | **₩5,000** | **₩17,595** | **약 271억** | **▲ 9.8%** |

### 최종 제안: 이중 요금제 (내국인 ₩5,000 / 외국인 ₩17,500)

- **외국인 ₩17,500**: 내/외국인 동일 탄력성(-0.3)에서 도출한 최적가. Step 1 회귀분석 결과($12.20)와 수렴하며, 별도 가정이 없어 모델 강건성이 높음
- **내국인 ₩5,000**: 방문객의 94.8%가 내국인이므로 내국인 가격이 총 방문객에 결정적 영향. 국내 유료 문화시설(경복궁 3,000원, MMCA 전시당2,000원) 대비 NMK 규모를 감안하여 설정
- **이중 요금제 근거**: 단일 ₩17,500은 수익 최대이나 방문객 34.8% 감소. 감성분석에서도 고가 유료 박물관일수록 부정 비율 급증이 확인되어, 공공성과 만족도 유지를 위해 이중 요금제 도입을 제안
